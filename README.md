# Hola Mundo en PHP

Este proyecto es una prueba básica para verificar el funcionamiento de PHP en un entorno Linux.

## 📝 Descripción
Un script simple que muestra un saludo personalizado en la terminal de Ubuntu.

## 💻 Entorno y Tecnologías
* **Lenguaje:** PHP
* **Editor:** Visual Studio Code
* **Sistema:** Ubuntu (VirtualBox)

## 🛠️ Instalación
Si no tienes PHP, instala el paquete necesario con este comando en tu terminal de Ubuntu:
```bash
sudo apt update && sudo apt install php-cli

## 🏃 Pasos para la Ejecución
📂 Abre Visual Studio Code en tu Ubuntu.

📄 Crea el archivo hola_mundo.php.

💾 Pega este código:

PHP

<?php
echo "Hola Mundo, soy Diego Tello";
?>
🖥️ En la terminal, ejecuta:

Bash

php hola_mundo.php
